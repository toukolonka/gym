# Getting Started

### Run these commands to run the application on your own machine

`git clone https://github.com/toukolonka/www-design.git`

`cd www-design`

`npm install`

`cd server`

`npm run dev`

`cd ../client`

`npm install`

`npm start`


### In addition to this you need an .env file which contains the following variables
#### DEV_MONGODB_URI
#### PORT
#### SECRET
